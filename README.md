# dotfiles

## Installing

Run `stow` to symlink everything or just select what you want

```sh
stow */ -t ~/ # Everything (the '/' ignores the README)
```

```sh
stow zsh # Just my zsh config
```

```sh
stow -nv */ -t ~/ # simulation
```

### tmux

Install plugins with `prefix + I`
